TECHNICAL DETAILS
================================================================================

COOKIE MANAGEMENT:
  Cookie Name: pferdewert_cookie_consent
  Values:
    • 'allow' → GA4 ✅ + DataFa.st ✅ (all analytics)
    • 'analytics_only' → GA4 ✅ + DataFa.st ✅ (modal: analytics ON)
    • 'necessary_only' → GA4 ❌ + DataFa.st ❌ (modal: analytics OFF)
  Expiry: 365 days (GDPR compliant)

ANALYTICS INTEGRATION:
  • GA4: Respect Consent Mode v2 (dynamic based on consent)
  • DataFa.st: Conditional loading (only if consent given)
  • MongoDB: Logs all consent decisions (audit trail)

RESPONSIVE DESIGN:

  • Mobile: Bottom-Sheet (native feel)
  • Touch targets: 44px+ (accessibility)
